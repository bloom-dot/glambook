# Images GlamBook

Dépose ici les visuels générés (Gemini/ChatGPT). Je m'occupe ensuite de :
- retirer automatiquement le petit logo Gemini (coin bas-droite) par recadrage/masque,
- découper les planches (grilles) en tuiles individuelles,
- optimiser et câbler les images dans la landing (hero + catégories).

## Noms de fichiers suggérés
- `hero.jpg`        → image principale du hero (maquillage en cours, format paysage)
- `hero-2.jpg`      → variante hero (optionnel)
- `portrait.jpg`    → portrait vertical (carte / mobile)
- `grid-categories.jpg` → planche 2×3 (Mariage / Soirée / Quotidien / Lèvres / Kit Pro…)
- `grid-eyes.jpg`   → planche yeux/looks (optionnel)

Astuce : garde les fichiers en JPG, largeur ≥ 1400 px pour le hero.
